## 二叉树中最长的连续序列

给定一个二叉树，你需要找出二叉树中最长的连续序列路径的长度。

请注意，该路径可以是递增的或者递减。例如，`[1,2,3,4]`和`[4,3,2,1]`都被认为合法的，
而路径`[1,2,3,4]`则不合法。另一方面，路径可以是子-父-子顺序，并不一定是父-子顺序。

示例1

```
输入：
    1
   / \
  2   3
输出：2
解释：最长的连接路径是[1,2]或者[2,1]
```

示例2

```
输入：
    2
   / \
  1   3
输出：3
解释：最长的连接路径是[1,2,3]或者[3,2,1]
```

注意：树上所有节点的值都在`[-1e7, 1e7]`范围内.
